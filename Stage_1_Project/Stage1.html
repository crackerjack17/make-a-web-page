Important Concepts

Lesson 1: The Basics of the Web and HTML

A Picture is worth a thousand words
  1. You have a computer
  2. You type a URL into a browser
  3. An HTTP request is sent to the corresponding server
  4. The server finds the page or document and sends it back to the requesting computer
  5. The user views the results (page/document) which is interpreted and displayed in a browser
HTML

HTML stands for Hypertext Markup Language also referred to as Hypertext Transfer Protocol. HTML documents form the majority of the content on the web. HTML documents contain text content which describes "what you see" and markup which describes "how it looks".

Tags and Elements

HTML is made up of elements that contain tags. The tags define the element such as boldness, italized, lists, hyperlinks to name a few.

Computers are only as smart as the person behind the curtain

Computers only function because a person has written, or coded, a program to make them work. When you make a mistake in your code, the computer will let you know it!

Inline vs. Block Elements

HTML elements are either Inline or Block. Block level elements normally start (and end) with a new line, when displayed in a browser. Inline elements are normally displayed without line breaks.

Lesson 2: Creating a Structured document with HTML

HTML, CSS and Javasript

HTML is the language used to code, CSS gives the coding style. CSS works on a system of rules, which select the elements you want to style, and then set values for different properties of the elements. You can change or add colors, backgrounds, font sizes and styles, and even position things on your web page in different places. Javascript allows you to add behaviour to your code, such as an animated element, validation of form data that's input and much more.

Structure of HTML

Developers use a "tree-like structure" that "nests" the element tags. This makes it easier to see where tags start and stop within a body of content. This proves especially helpful when editting contents.

HTML and boxes

As noted above in the structure of HTML, you will see that coding and content is structured in "boxes". In a sense, your content is comparmentalized. This is helpful for organization and readability of your code.

Text Editors

Text editors such as Stratchpaid.io, Codepen, etc. have some built in and automated editting features allowing the user to more easily create and have code validated. It's like an extra set of eyes on your code to make sure it displays as desired.

Lesson 3: Add CSS style and HTML

Box Sizing and Positioning

HTML elements are boxes and each box has 4 components: Margin; Padding; Border and the Box itself. There are two ways to size these componets, by pixel or percentage.

CSS

CSS stands for Cascading Style Sheets. You can use mulitple sheets in your coding. Cascading is the idea knows as inheritance and understanding requires abstract thinking. Some properties that can be inherited are color, font, and certain spacing.
